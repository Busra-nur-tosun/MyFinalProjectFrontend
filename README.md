 Final  Projesi
Bu proje kapsamında bir Eticaret sistemi backendini ASP.Net Core, frontendini ise Angular kullanarak simüle etmişti. Amacım bu simülasyonu gerçekleştirirken SOLID kod yazma kurallarına uyum ve n-Layered Architecture kullanarak proje ölçeklenebilirliğini katmanlaştırdı. Motivasyonum ise kodlama.io'da katılıyorum C# & Angular yazılım geliştirme kampı sağladı.
